Hacking
=======

To setup a local dev environment for hacking::

1. Create and activate a new virtual environment.
2. Clone the repo::

    $ git clone git://github.com/rlr/standup.git
    $ cd standup

3. Install required dependencies::

    $ pip install -r requirements.txt

4. Run the app::

    $ python app.py
